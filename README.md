# Flappy Tails
Flappy bird clone with Sonic aesthetics using [Gosu](https://github.com/jlnr/gosu) following [Tom Dalling's Gosu tutorial](https://www.youtube.com/watch?v=QtIlyU2Br3o).


## Instructions
```
bundle install
bundle exec ruby game.rb
```

## Keys
```
Space -> Jump
Escape -> Quit
```
